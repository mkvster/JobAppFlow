# CLI Admin DB

## Description

Create a command-line admin tool that can connect to the database and execute admin commands.
For now, implement a single command for registering a user.
The tool should be able to register the local development users `admin` or `owner`, `tester`, and `demo`.

## Acceptance

- A CLI admin tool exists.
- The tool can connect to the database.
- The tool supports a user registration command.
- The local database can be populated with `admin` or `owner`, `tester`, and `demo` users.
- Usage documentation is available.
