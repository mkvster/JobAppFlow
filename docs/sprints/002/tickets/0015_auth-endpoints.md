# Auth Endpoints

## Description

Implement the authentication endpoints `/api/auth/login`, `/api/auth/refresh`, `/api/auth/logout`, and `/api/auth/me`.
Verify the endpoints locally and deploy the result to Azure production.

## Acceptance

- `/api/auth/login` is implemented.
- `/api/auth/refresh` is implemented.
- `/api/auth/logout` is implemented.
- `/api/auth/me` is implemented.
- The endpoints are verified on the local machine.
- The endpoints are deployed to Azure production.
