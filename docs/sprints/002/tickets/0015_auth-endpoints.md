# Auth Endpoints

## Description

Implement the authentication endpoints `/api/v1/auth`:

`/login` 
`/refresh` 
`/logout`
`/me`

Verify the endpoints locally and deploy the result to Azure production.

## Acceptance

- `/api/v1/auth/login` is implemented.
- `/api/v1/auth/refresh` is implemented.
- `/api/v1/auth/logout` is implemented.
- `/api/v1/auth/me` is implemented.
- The endpoints are verified on the local machine.
- The endpoints are deployed to Azure production.
