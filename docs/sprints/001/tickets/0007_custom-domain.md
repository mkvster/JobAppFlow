# Custom Domain

## Description

Connect the purchased custom domain to the production environment and verify that the application is publicly accessible through it.
The setup should include the required DNS, HTTPS, and Azure configuration to make the site available on the custom domain.
The final check should confirm that the frontend opens successfully and that the backend is reachable through the public production setup.

## Acceptance

- The custom domain is connected to the production environment.
- DNS and Azure configuration are in place for public access.
- HTTPS works on the custom domain.
- The frontend opens successfully through the custom domain.
- The backend is reachable through the public production setup.
