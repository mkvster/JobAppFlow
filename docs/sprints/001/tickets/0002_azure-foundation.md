# Azure Foundation

## Description

Set up the base Azure environment for the first production delivery of JobAppFlow.

Create the following Azure resources manually:

- Azure Resource Group;
- Azure App Service Linux for the backend API;
- Azure Static Web Apps for the frontend;

## Acceptance

- The required Azure resources are created and configured manually.
- The environment includes the core production hosting pieces needed for delivery.
- The Azure environment is ready to receive a future backend and frontend setup.
