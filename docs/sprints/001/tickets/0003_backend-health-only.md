# Backend Health Only

## Description

Create a minimal backend solution that can run and exposes `GET /health`.
The solution should include at least an API project and the basic structure needed to start the service locally.
The solution should also include a minimal xUnit or MSTest project for validating the health endpoint.

## Acceptance

- A backend solution exists.
- The API project starts successfully.
- The backend starts successfully.
- `GET /health` responds successfully.
- A test project validates `GET /health`.
